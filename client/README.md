# MERN Shop

A full-stack e-commerce application built with the MERN stack.

## Tech Stack

- React
- Node.js
- Express
- MongoDB
- Vite

## Project Structure

client → React frontend

server → Node.js + Express API

## Installation

### Clone repository
git clone https://github.com/mahdighobadi73/shop.git

### Install dependencies

Frontend
cd client

npm install

### Run project

Backend
cd server

npm run dev

Frontend
cd client

npm run dev
